# Cura — Технічне завдання

> Персональний помічник з харчування: підбір раціону з урахуванням індивідуальних потреб людини.

---

## Мета проєкту

Допомогти користувачу харчуватися збалансовано — не просто показати цифри БЖВ, а скласти реальний план харчування на день з рецептами та рекомендаціями.

---

## Фази розробки

### Фаза 1 — MVP (поточна версія)

#### 1.1 Профіль користувача
- Стать (чоловік / жінка)
- Вік
- Вага (кг), Зріст (см)
- Рівень фізичної активності:
  - Мінімальний (сидячий спосіб життя)
  - Легкий (1-3 тренування на тиждень)
  - Середній (3-5 тренувань)
  - Високий (6-7 тренувань)
  - Екстремальний (спортсмени)
- Мета: схуднення / підтримка ваги / набір маси

#### 1.2 Розрахунок калорійності та БЖВ
- Формула Міффліна-Сан Жеора для базового метаболізму (BMR)
- Корекція на активність (TDEE)
- Розподіл макронутрієнтів залежно від мети:
  - Схуднення: білки 30%, жири 30%, вуглеводи 40%
  - Підтримка: білки 25%, жири 30%, вуглеводи 45%
  - Набір маси: білки 25%, жири 25%, вуглеводи 50%

#### 1.3 Адаптація для жінок (менструальний цикл)
- Фаза 1 (менструальна, дні 1-5): -10% калорій, більше заліза
- Фаза 2 (фолікулярна, дні 6-13): стандарт + вуглеводи
- Фаза 3 (овуляторна, дні 14-17): +5% білків
- Фаза 4 (лютеїнова, дні 18-28): +150-200 ккал, більше магнію

#### 1.4 Денний раціон
- Розподіл на прийоми їжі (сніданок / обід / вечеря / перекус)
- Для кожного прийому:
  - Назва страви
  - Склад (продукти + кількість у грамах)
  - КБЖВ страви
  - Метод приготування (варіння / запікання / на парі / свіже)
  - Короткий рецепт (кроки)
- Підсумок за день: загальне КБЖВ

---

### Фаза 2 — Розширення (майбутнє)

#### 2.1 Обмеження та алергії
- Список продуктів, які користувач не вживає
- Алергени (глютен, лактоза, горіхи, морепродукти тощо)
- Вегетаріанство / веганство / інші дієти
- Автоматична заміна страв у раціоні

#### 2.2 Раціон з холодильника
- Користувач вказує продукти, що є вдома
- Система підбирає страви з наявних продуктів
- Список того, що варто докупити для збалансованого тижневого раціону

#### 2.3 Поради при відновленні
- Відновлення після хвороби
- Відновлення після тренувань
- Постопераційне харчування (довідково)

#### 2.4 Тижневий план
- Генерація плану на 7 днів
- Різноманітність страв (без повторів)
- Список покупок на тиждень

## Технічний стек (MVP)

| Частина | Технологія |
|---------|-----------|
| Frontend | HTML + CSS + Vanilla JS |
| Розрахунки | JS (формули BMR/TDEE) |
| Рецепти | Статична база даних (JSON) |
| Зберігання профілю | localStorage |
| Деплой | GitHub Pages |

> На майбутнє: бекенд (Node.js/Python) + база рецептів + AI-підбір страв

---

## Екрани / Кроки (MVP)

```
1. Вітання → Почати
2. Анкета: стать, вік, вага, зріст, активність, мета
3. (для жінок) День циклу
4. Результат: КБЖВ норма + пояснення
5. Денний раціон: 4 прийоми їжі з рецептами
6. Можливість "Оновити раціон" (перегенерувати страви)
```

---

## База рецептів (MVP — мінімум)

По 3-5 варіанти для кожного прийому їжі:
- **Сніданки**: вівсянка, омлет, грецький йогурт з ягодами, тости з авокадо
- **Обіди**: курка з рисом, суп, греча з овочами, салат з тунцем
- **Вечері**: риба на парі, індичка з овочами, сочевиця, яйця з брокколі
- **Перекуси**: фрукти, горіхи, хумус, протеїновий бар

---

## Що НЕ входить в MVP
- Реєстрація / авторизація
- Синхронізація між пристроями
- Трекер прийомів їжі (що фактично з'їв)
- Інтеграція з фітнес-додатками
